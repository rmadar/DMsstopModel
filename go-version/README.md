# Installation

Once you have installed [Go](https://golang.org/doc/install):

```sh
$> go get github.com/rmadar/DMsstopModel
```

And voila. Everything has been installed (including our dependencies.)

# Running it

To run it:

```sh
$> $GOPATH/bin/DMsstopModel
$> open go-*.png
```
